СОЛНЦАНЕТ CRM v68 SMS QUEUE FULL

Что это за сборка:
- Полная чистая сборка сайта/CRM для загрузки в корень GitHub.
- Внутри уже есть: index.html, zapis.html, admin.html, assets, functions, docs, .github/workflows.
- Старые патчи v62-v67 отдельно подключать не нужно.

Главное в v68:
1. Добавлена полноценная вкладка SMS.
2. В SMS-панели видно очередь автоматических SMS:
   - подтверждение записи;
   - напоминание за день;
   - напоминание за 2 часа.
3. В SMS-панели можно:
   - изменить время конкретной SMS;
   - отменить конкретную SMS;
   - вернуть отменённую SMS в очередь;
   - отправить конкретную SMS сейчас;
   - вручную запустить проверку очереди.
4. Перенос и благодарность с отзывом НЕ отправляются автоматически. Они остаются только ручными кнопками в карточке заявки.
5. Для направления Авто скрываются и очищаются: Итоговый м², Адрес, общий Материал/Плёнка. Материал остаётся только в строках услуг.

Как загрузить:
1. Удалите старые файлы в GitHub или создайте чистый репозиторий.
2. Распакуйте архив SOLNCANET_CRM_V68_SMS_QUEUE_FULL.zip.
3. Загрузите СОДЕРЖИМОЕ папки SOLNCANET_CRM_V68_SMS_QUEUE_FULL в корень репозитория.
   В корне должны лежать сразу index.html, admin.html, zapis.html, assets, functions, docs.
4. Дождитесь деплоя Cloudflare Pages.
5. Откройте admin.html, войдите в CRM, нажмите Проверка -> Проверить Cloudflare / NocoDB / SMS.
6. Откройте вкладку SMS -> Обновить очередь.

Переменные Cloudflare Pages:
ADMIN_PASSWORD=ваш_пароль_админки
NOCODB_API_URL=https://ваш-nocodb
NOCODB_TABLE_ID=ID_таблицы_заявок
NOCODB_API_TOKEN=токен
SMS_ENABLED=1
SIGMASMS_USERNAME=логин
SIGMASMS_PASSWORD=пароль
SIGMASMS_SENDER=SOLNCANET
SMS_CRON_SECRET=длинный_секрет
SMS_TIMEZONE_OFFSET_MINUTES=300

Переменные GitHub Actions:
SOLNCANET_SITE_URL=https://адрес-сайта.pages.dev
SMS_CRON_SECRET=тот_же_секрет_что_в_Cloudflare

Важно:
- SMS-очередь хранит свои служебные отметки внутри поля «Комментарий администратора» в скрытом маркере.
- В интерфейсе и выгрузках этот служебный маркер скрывается.
- Новые отдельные колонки NocoDB для очереди не требуются.
