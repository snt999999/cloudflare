# СОЛНЦАНЕТ v9 1. Во вкладке «Монтажники» добавлена история работ по каждому монтажнику. - Клик по имени или кнопке «История работ» открывает детальную карточку монтажника. - Внутри есть поиск по клиенту, телефону, адресу, услуге, объекту и комментариям. - Добавлены фильтры по датам, статусу, услуге и м². - Каждый объект можно открыть и редактировать через существующую карточку заявки. - Показывается сводка: количество работ, м², начислено, средняя ставка. 2. На клиентской странице заменены повторяющиеся изображения услуг. - Пересозданы отдельные изображения для каждой услуги. - Добавлена отдельная hero-картинка, чтобы первый экран не повторял карточку услуги. 3. Сохранены функции v8: заявки, Web3Forms, NocoDB, корзина отмен, история изменений и зарплатный отчёт.
