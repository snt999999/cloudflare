V24 — закреплённая панель, поиск по всей админке, карточка клиента

Добавлено:
1. Закреплённая верхняя панель в админке.
   - Быстрая запись всегда под рукой.
   - Обновить данные всегда под рукой.
   - Быстрый переход к зарплатному отчёту.
   - Ссылка на страницу записи.

2. Единый быстрый поиск по всей админке.
   Поиск ищет сразу по:
   - заявкам;
   - клиентам;
   - компаниям;
   - телефонам;
   - адресам;
   - услугам;
   - статусам;
   - файлам.

3. Карточка клиента как мини-CRM.
   В карточке клиента видно:
   - ФИО;
   - компанию;
   - телефон;
   - список услуг;
   - все адреса / объекты;
   - всю историю заявок;
   - м²;
   - выполненные заявки;
   - файлы клиента;
   - комментарии.

4. Из карточки клиента можно сразу создать новую заявку этому клиенту.

Проверено:
- admin.js проходит проверку синтаксиса;
- site.js проходит проверку синтаксиса;
- Cloudflare Functions проходят проверку синтаксиса.
