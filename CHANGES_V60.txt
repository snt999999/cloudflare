# v60 — исправление отчёта по авто

- Стоимость услуги по умолчанию берётся из таблицы `Книга1.xlsx` (колонка «Стоимость»).
- В отчёте по авто сумма заказа считается из услуг, сохранённых в заявке.
- Оплата сотруднику считается отдельно по прайсу `prais_zp_avto_uslugi_solncanet(4).xlsx` (колонка «Оплата сотруднику»).
- Поддержаны старые и новые форматы хранения авто-услуг: price/cost/amount/Стоимость/Сумма.
- Если услуга не найдена в прайсе оплаты — в детализации будет пометка «не найдено в прайсе».
- Кеш приложения обновлён до v60.
