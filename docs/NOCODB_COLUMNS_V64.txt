Обязательные колонки в таблице заявок NocoDB для направления "Авто":

1. Направление
   Рекомендуемый тип: SingleSelect или Single line text
   Значения: Авто, Архитектура

2. Авто
   Рекомендуемый тип: Single line text
   Пример: Tiguan, Solaris 2015, BMW X5

3. Авто услуги
   Рекомендуемый тип: Long text
   Формат хранения: JSON
   Пример:
   [{"name":"Тонирование задней полусферы","material":"Xpel 5","price":"10000"}]

4. Общая стоимость
   Рекомендуемый тип: Number, Decimal или Single line text
   Пример: 22000

Не обязательные для авто:
- Итоговый м2 — для авто скрывается и очищается
- Адрес — для авто скрывается и очищается
- Пленка — общий материал для авто больше не нужен
