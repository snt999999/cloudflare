Рекомендуемые колонки NocoDB для CRM v68

Минимум:
- Id
- Дата записи
- Время записи
- Статус
- Направление
- Имя клиента
- Телефон
- Компания
- Ответственный
- Комментарий администратора

Архитектура:
- Адрес
- Услуга
- м2
- Итоговый м²
- Плёнка
- Пленка
- Материал

Авто:
- Авто
- Авто услуги
- Общая стоимость

SMS-очередь:
- отдельные колонки не требуются;
- служебный лог хранится в поле «Комментарий администратора».
