СОЛНЦАНЕТ сайт v8

Что изменено:
1. Клиентский сайт:
   - Убраны дублирующиеся изображения услуг: у каждой услуги теперь свой отдельный файл-картинка.
   - Обновлены тексты услуг по смыслу страниц солнцанет.рф: авто, балконы/окна, атермальная тонировка, бронирование стекол, фары, офисные перегородки, PPF, витрины.
   - Сохранена отправка заявки через create-public-lead: NocoDB + Web3Forms при наличии WEB3FORMS_ACCESS_KEY.

2. Раздел Клиенты:
   - Добавлены фильтры по дате, услуге, статусу, плёнке/материалу.
   - Добавлен быстрый поиск по имени, телефону, услуге, адресу и комментариям.
   - Добавлена статистика: количество клиентов, заявок, м², повторных клиентов.

3. Раздел Объекты:
   - Добавлены фильтры по дате, услуге, статусу, монтажнику, диапазону м².
   - Добавлен интерактивный поиск по адресу, объекту, клиенту, телефону и комментариям.
   - Добавлена статистика по объектам, объёму, выполненным и объектам в работе.

4. Раздел Монтажники:
   - Добавлены фильтры по дате, статусу, услуге и поиску.
   - Добавлена сводка начислений, доплат, авансов и итогов.
   - Добавлена кнопка быстрого открытия отчёта по зарплате.

5. Зарплатный отчёт:
   - Сделан матричный расчёт по ставкам из предоставленной таблицы:
     Пахнев Никита: 500 / 300 / 300 / 300 / 300
     Шолохов Андрей: 500 / 300 / 250 / 250 / 250
     Петухов Дмитрий: 400 / 200 / 200 / 200 / 200
     Зинченко Роман: 400 / 200 / 200 / 200 / 200
     Кислов Никита: 500 / 250 / 200 / 200 / 200
   - Ставка автоматически выбирается по количеству монтажников на объекте.
   - Основной режим: полный м² объекта каждому монтажнику, ставка зависит от количества монтажников.
   - Добавлены альтернативные режимы: делить м² поровну; полный м² каждому по ставке 1 человека.
   - Добавлена выгрузка настоящего Excel-compatible .xls файла со сводкой, детализацией, ставками и параметрами отчёта.
   - CSV оставлен как запасной формат.

6. История и корзина:
   - Сохранены отмена/корзина и история изменений из v7.

Важно:
- Для отправки письма через Web3Forms в Cloudflare Pages нужно добавить переменную WEB3FORMS_ACCESS_KEY.
- Для работы заявок в админке нужны ADMIN_PASSWORD, NOCODB_TOKEN и NOCODB_ENDPOINT.
