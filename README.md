# СОЛНЦАНЕТ — исправленная версия сайта и админ-панели

Исправления:
- логотип заменён на реальный PNG, который был загружен в чат;
- генеративные картинки удалены;
- используются реальные фото с сайта солнцанет.рф и прямые ссылки на изображения сайта;
- услуги идут вперемешку: авто / архитектура / авто / архитектура;
- у каждой услуги рабочая ссылка на соответствующую страницу основного сайта;
- шапка, кнопки, форма, онлайн-запись и админ-панель приведены в рабочий вид.

Загрузка:
1. Распаковать архив.
2. Загрузить содержимое папки в корень GitHub-репозитория.
3. В корне должны быть: index.html, zapis.html, admin.html, assets, functions, README.md.
4. Дождаться Cloudflare Deploy.
5. Открыть сайт и нажать Ctrl+F5.

Проверка:
- /test
- /nocodb-test
- /admin.html
- /cal-nocodb-v5

Переменные Cloudflare:
- NOCODB_TOKEN
- ADMIN_PASSWORD
