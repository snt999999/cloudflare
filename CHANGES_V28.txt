СОЛНЦАНЕТ v28 — SMS history UI + Prostor status card

Что изменено:
1. Полностью переработан раздел SMS-уведомлений: статистика, история, поиск, фильтры, более читабельная таблица.
2. Добавлена карточка SMS-отправления по клику на строку/кнопку «Карточка».
3. В карточке отображаются: клиент, телефон, ID заявки, ID Prostor/smscId, Client ID, статус доставки, текст SMS, ответ сервиса и ошибка.
4. Из карточки можно проверить статус Prostor, скопировать ID, открыть заявку, отправить запланированное SMS сейчас или отменить SMS.
5. Поле телефона в тестовой отправке автоматически подставляет +7.
6. Исправлена/усилена кнопка «Обновить историю»: добавлен no-cache, блокировка кнопки на время загрузки, расширенная ошибка и автообновление при входе в раздел.
7. Отправка из очереди больше не создаёт дубль в SMS-истории: существующая запись обновляется ID Prostor и ответом сервиса.
8. Добавлены статусные метки и расшифровка статусов доставки Prostor.
9. Улучшен визуал раздела: ровные карточки, читаемые таблицы, аккуратные статусы, моноширинный ID, адаптивность.
