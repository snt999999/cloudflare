V22 — ручной импорт из Google Календаря

- Добавлена вкладка «Импорт календаря» в админке.
- Добавлена Cloudflare Function /calendar-events.
- Добавлен Apps Script google-calendar-import-appsscript.gs для чтения Google Календаря.
- События показываются карточками с датой, временем, названием, телефоном, компанией, услугой и адресом.
- Добавлена авторазборка событий: телефон, компания, услуга, адрес, м².
- Добавлены кнопки: Создать заявку, Проверить / заполнить, Не работа, Вернуть, Открыть в Google.
- Личные события можно скрывать локально, чтобы они не мешались.
- Созданные события защищены от дублей через Cal Booking ID = gcal-<eventId>.
- Можно быстро открыть уже созданную заявку из календарной карточки.
