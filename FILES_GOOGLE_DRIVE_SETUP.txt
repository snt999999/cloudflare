Подключение бесплатного хранения файлов через Google Drive + Apps Script

1. Создайте папку на Google Drive, например:
   SOLNCANET_FILES

2. Откройте папку и скопируйте ID из ссылки.
   Пример ссылки:
   https://drive.google.com/drive/folders/1AbCdEfG...
   ID папки — это часть после /folders/

3. Откройте Google Apps Script:
   https://script.google.com/

4. Создайте новый проект.

5. Вставьте туда код из файла:
   google-drive-files-appsscript.gs

6. В начале файла замените:
   FILES_FOLDER_ID = 'PASTE_GOOGLE_DRIVE_FOLDER_ID_HERE'
   на ID вашей папки Google Drive.

7. Придумайте секретный токен и замените:
   UPLOAD_TOKEN = 'PASTE_SECRET_TOKEN_HERE'
   например:
   UPLOAD_TOKEN = 'solncanet-secret-2026'

8. Опубликуйте Apps Script:
   Deploy → New deployment → Select type: Web app

   Обязательно:
   Execute as / Запуск от имени: Me / Меня
   Who has access / У кого есть доступ: Anyone / Все

9. Нажмите Deploy.
   Google попросит разрешения на доступ к Drive — разрешите.

10. Скопируйте Web App URL.
    Нужна именно ссылка, которая заканчивается на:
    /exec

    НЕЛЬЗЯ вставлять ссылку редактора script.google.com/.../edit
    НЕЛЬЗЯ вставлять ссылку /dev
    НУЖНА именно Web App ссылка /exec

11. В Cloudflare Pages добавьте переменные:

   GOOGLE_DRIVE_UPLOAD_URL=ссылка_Web_App_которая_заканчивается_на_/exec
   GOOGLE_DRIVE_UPLOAD_TOKEN=тот_же_секретный_токен

12. Важно: после добавления переменных сделайте Redeploy проекта в Cloudflare Pages.

13. В админке откройте вкладку «Файлы» и нажмите:
   Проверить Google Drive

Если появляется ошибка «Apps Script вернул не JSON», причины почти всегда такие:

1) В GOOGLE_DRIVE_UPLOAD_URL вставлена неправильная ссылка.
   Должна быть ссылка Web App, заканчивающаяся на /exec.

2) Web App опубликован не для всех.
   Нужно: Execute as Me, Who has access Anyone.

3) После правки Apps Script не опубликована новая версия.
   Нужно: Deploy → Manage deployments → Edit → Version: New version → Deploy.

4) В Apps Script не заменён FILES_FOLDER_ID.

5) Токен в Apps Script и GOOGLE_DRIVE_UPLOAD_TOKEN в Cloudflare не совпадают.

Ограничение бесплатной схемы:
- Большие файлы лучше грузить партиями.
- Для Google Apps Script безопаснее загружать за один раз до 20–25 МБ.
