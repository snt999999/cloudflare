V16 — предпросмотр файлов в админке

Что добавлено:
1. Во вкладке «Файлы» клик по названию файла, миниатюре или кнопке «Посмотреть» открывает файл прямо в админ-панели.
2. В карточке заявки блок «Файлы этой заявки» работает так же: файл можно открыть в модальном окне без поиска в Google Drive.
3. Фото показываются как изображение внутри админки.
4. PDF, видео и документы Google Drive открываются через встроенный предпросмотр Google Drive.
5. Оставлены отдельные действия: «Drive» — открыть оригинал в Google Drive, «Скачать» — скачать файл, «Удалить» — удалить файл и убрать его из заявки.
6. Добавлены стили для окна предпросмотра файлов.

Важно:
- Для предпросмотра Google Drive файл должен иметь доступ «Anyone with the link / Все, у кого есть ссылка». Apps Script включает этот доступ автоматически.
- Если файл большой или Google Drive ещё обрабатывает видео, предпросмотр может появиться не сразу. В этом случае нажмите «Drive».
