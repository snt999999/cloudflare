V12 — подключение сервиса файлов Cloudflare R2

Что добавлено:
1. Вкладка «Файлы» теперь не декоративная: добавлена загрузка файлов в Cloudflare R2.
2. Файлы можно привязывать к конкретной заявке.
3. В карточке заявки появился блок «Файлы этой заявки».
4. Можно загружать несколько файлов сразу: фото, видео, PDF, документы.
5. Можно скачать файл из админки.
6. Можно удалить файл из R2.
7. Добавлен поиск по файлам: номер заявки, клиент, телефон, адрес, услуга, название файла.
8. Добавлена фильтрация по типу файла: фото, видео, PDF, документы.

Новые Cloudflare Pages Functions:
- /upload-file
- /list-files
- /get-file
- /delete-file

Что нужно настроить в Cloudflare:
1. Создать R2 bucket, например solncanet-files.
2. В Pages проекте открыть Settings → Bindings.
3. Add binding → R2 bucket.
4. Variable name: FILES_BUCKET.
5. Выбрать bucket solncanet-files.
6. Сохранить и сделать redeploy проекта.

Важно:
Файлы защищены паролем админки через заголовок x-admin-password. Прямые публичные ссылки на файлы не создаются.
