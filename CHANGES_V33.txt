СОЛНЦАНЕТ v33 — SMS.ru only

- Полностью убрана поддержка старого SMS-сервиса из интерфейса и серверных функций SMS.
- SMS отправляются только через SMS.ru API.
- Используется имя отправителя SOLNCANET через переменную SMSRU_SENDER.
- История SMS сохраняет ID SMS.ru, статус доставки, ответ сервиса и баланс после отправки.
- Проверка статуса SMS работает через /sms-status и API SMS.ru.
- Очередь автоматических SMS и Cron продолжают работать через NocoDB.
