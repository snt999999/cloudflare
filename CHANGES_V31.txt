V31 — календарь, файлы, Google Calendar и мобильное меню

1. Календарь: теперь день выбирается нажатием по всей области ячейки, а не только по цифре.
2. Мобильное меню: вкладки открываются боковой шторкой, есть затемнение и кнопка закрытия.
3. Вкладка «Файлы»: теперь показывает только заявки, где реально есть прикрепленные файлы.
4. Файлы: после загрузки файл сохраняется в поле «Файлы» заявки в NocoDB, поэтому связь файл → заявка не теряется.
5. При открытии файла из карточки заявки или вкладки «Файлы» используется сохранённая ссылка Google Drive.
6. Google Calendar: после загрузки/удаления файлов событие календаря автоматически обновляется.
7. В описании события Google Calendar добавляется блок «Файлы Google Drive» со ссылками на файлы.
8. В Apps Script добавлена попытка прикреплять файлы к событию как вложения Google Calendar через Advanced Calendar API. Если Advanced Calendar API не включён, ссылки всё равно будут добавлены в описание события.
9. Обновлён service-worker до v31, чтобы мобильное приложение подтянуло свежие файлы.
